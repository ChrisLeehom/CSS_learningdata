做程序员的五大好处

第一个就是相对比较高的薪资，IT行业最近几年超过金融行业，成为了收入最高的行业。

第二是有更公平的竞争环境，也就是不看你的长相、不看你的背景，你的能力决定一切。

第三是能力可以不断的提升，不断接触到新的技术。随着你工作中实战项目越来越多，你编码的能力也会越来越强。而且你基本上是一边拿着高工资，一边在提升自己。

第四是无论你是在金融领域做开发，还是在医疗领域等等，慢慢经过时间的沉淀，你在这个领域里也加强了自己的核心竞争力。

第五是成就感，小到解决一个bug，大到开发出让领导、客户满意的平台或者软件，你都会收获满满的成绩感。
